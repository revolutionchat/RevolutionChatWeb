Revolution Python API

Welcome to our API. We currently have a NodeJS and a Python library. Enjoy and be happy coding!

Install our Python library:

$ pip install revolution.py pip3 install revolution.py

    .bot
    The original bot object that the class came from.
    Returns: [ BotApplication ]
    .data
    READ-ONLY; The server raw data.
    Returns: [ { "name": ..., "id": ..., "abbr": ..., "imgurl": ..., "color": ..., "channels": [ ... ], "roles": [ ... ], "emojis": [ ... ], "emotes": [ ... ], "members": [ ... ], ... } ]
    .name
    READ-ONLY; The server name.
    Returns: [ str ]
    .id
    READ-ONLY; The server ID.
    Returns: [ str \ int ]
    .abbr
    READ-ONLY; The server abbreviation.
    Returns: [ str ]
    .abbrevation
    READ-ONLY; The server abbreviation.
    Returns: [ str ]
    .icon
    READ-ONLY; The server icon url. If it starts with /, add https://revolution-web.repl.co in front of the url to get the full image.
    Returns: [ str ]
    .color
    READ-ONLY; The server color. Usually in hex, rgb is secondary.
    Returns: [ str ]
    .channels
    READ-ONLY; The server channels.
    Returns: [ map(..., Channel, ...) ]
    .roles
    READ-ONLY; The server roles.
    Returns: [ map(..., Role, ...) ]
    .emojis
    NOT IMPLEMENTED; READ-ONLY; The server emojis.
    Returns: [ map(unknown) ]
    .emotes
    NOT IMPLEMENTED; READ-ONLY; The server emotes.
    Returns: [ map(unknown) ]
    .members
    READ-ONLY; The server members.
    Returns: [ map(..., Member, ...) ]
    .partial
    READ-ONLY; Tells you if it is a partial server or not, usually for this class, it returns False.
    Returns: [ boolean(True, False) ]

    .author
    The author of the message sent.
    Returns: [ Member ]
    .author_id
    READ-ONLY; The author's ID.
    Returns: [ str \ int ]
    .server_id
    READ-ONLY; Returns the parent ID of the channel the message was sent in.
    Returns: [ str \ int ]
    .server
    The server the message was sent in.
    Returns: [ Server(*) ]
    .content
    READ-ONLY; The content of the message.
    Returns: [ str ]
    .channel
    READ-ONLY; The channel the message was sent in.
    Returns: [ Channel(*) ]
    .channel_id
    READ-ONLY; Returns the channel ID from the Channel the message was sent in.
    Returns: [ str \ int ]
    .message_id
    READ-ONLY; Returns the message ID.
    Returns: [ str \ int ]
    .id
    READ-ONLY; Returns the message ID.
    Returns: [ str \ int ]
    .timestamp
    READ-ONLY; Returns the epoch timestamp for when the message was sent.
    Returns: [ int ]
    .bot
    READ-ONLY; Retrieves the user's presence and sees if they're a bot or not.
    Returns: [ boolean(True, False) ]
    .staff
    READ-ONLY; Retrieves the user's presence and sees if they're a staff member or not.
    Returns: [ boolean(True, False) ]
